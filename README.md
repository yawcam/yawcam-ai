### Docker Deployment (Community Contribution)

There are community-maintained Docker images and example docker-compose configurations
available for running Yawcam Ai inside a container with persistent storage and optional
GPU acceleration.

Images:
- CPU build: `devprincekumar/yawcam-ai`
- CUDA enabled build: `devprincekumar/yawcam-ai-cuda`

Example docker-compose files are located in this repository under `/docker`.

Full documentation and ongoing updates:
https://github.com/855princekumar/yawcam-ai-dockerized

> Note: These Docker artifacts are **not official releases** of the Yawcam Ai project, but
> contributed by the community for convenience and testing purposes.
