### Docker (community-maintained)

There is a community-maintained Docker setup for running Yawcam Ai in a container, including:

- CPU-only image
- CUDA-enabled image (GPU acceleration where available)
- Example docker-compose files
- Persistent configuration, models, logs and recordings

> Note: These images are not officially maintained by the Yawcam Ai project.  
> They are provided by the community for convenience.

Links:

- Docker images (maintained by @855princekumar):  
  - CPU: `devprincekumar/yawcam-ai`  
  - GPU: `devprincekumar/yawcam-ai-cuda`
- Docker setup and documentation:  
  - https://github.com/855princekumar/yawcam-ai-dockerized
- Example docker-compose files in this repo:  
  - `docker/CPU/docker-compose.yml`  
  - `docker/GPU/docker-compose.yml`
